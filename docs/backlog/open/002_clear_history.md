# Clear History Feature

## What
Add a "Clear History" button to the web interface that allows users to wipe the visible download history.

## Why
Users may want to reset their view of the history without deleting the actual files or the archive metadata, or simply to declutter the interface.

## Acceptance Criteria
- [ ] A "Clear History" button is visible in the UI.
- [ ] Clicking the button clears the list of recent jobs in the UI.
- [ ] The history file on the server is reset or cleared.
- [ ] The actual downloaded files are NOT deleted.
