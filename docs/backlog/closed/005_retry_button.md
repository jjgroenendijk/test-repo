# Add Retry Button

## What
Add a "Retry" button to each history item in the dashboard.

## Why
Users may want to retry a failed download or re-download content with the same settings without manually re-entering the URL and mode.

## Acceptance Criteria
- [ ] Each history item displays a "Retry" button.
- [ ] Clicking "Retry" populates the "Video URL", "Mode", and "Playlist" fields in the form with the values from the history record.
- [ ] Clicking "Retry" scrolls the page to the top (to the form).
