# Download Files from History

## What
Add a download link next to each file in the history list.

## Why
Users currently see file paths but cannot access the files directly from the web interface. This feature allows users to retrieve the archived files easily.

## Acceptance Criteria
- [ ] Each file in the history list is a clickable link.
- [ ] Clicking a link initiates a file download.
- [ ] Files are served securely from the data directory.
- [ ] Only files within the downloads directory can be accessed.
